## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
npm install
```

## Running the app

```bash
# development
npm run start

# watch mode
npm run start:dev

# production mode
npm run start:prod
```

## Test

```bash
# unit tests
npm run test

# e2e tests
npm run test:e2e

# test coverage
npm run test:cov
```

### Debugging e2e tests

```bash
# add a break point in the e2e test file or in server code
yarn test:e2e:debug
# in VS Code, Ctrl+Shift+P -> `Debug: Attach to Node Process` -> select `...yarn.js test:e2e:debug`
# hit F5 / F10
```
